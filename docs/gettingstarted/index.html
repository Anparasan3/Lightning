Getting started

Web platform

Create a Lightning build

git clone https://github.com/WebPlatformForEmbedded/Lightning.git
cd Lightning
npm i

You can find the distributable in dist/lightning-wpe.js.

Usage

Create a file src/MyApp.js:

export default class MyApp extends lng.Application {
    static _template() {
        return {
            Text: {text: {text: "hello world"}}
        }
    }
}

Add an index.html file:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8" />
    <script src="./lightning-web.js"></script>
</head>
<body>
    <script type="module">
        import MyApp from "./src/MyApp.js";
        
        window.onload = function() {
            const options = {stage: {w: 1920, h: 1080, clearColor: 0xFF000000}}
            options.keys = {
                38: "Up",
                40: "Down",
                37: "Left",
                39: "Right",
                13: "Enter",
                9: "Back",
                8: "Back",
                93: "Back",
                174: "Back",
                175: "Menu",
                83: "Search"
            };

            const app = new MyApp(options);
            document.body.appendChild(app.stage.getCanvas());
        }
    </script>
</body>
</html>

Node.JS

Add a new package.json for your project:

{
  "author": "Your name",
  "name": "your-app",
  "version": "1.0.0"
}

Run npm i https://github.com/WebPlatformForEmbedded/Lightning-node.git --save

Your application must extend lng.Application:

import lng from "wpe-lightning-node";

class MyApp extends lng.Application {
    static _template() {
        return {
            Text: {text: {text: "hello world"}}
        }
    }
}

const options = {stage: {w: 900, h: 900, clearColor: 0xFF000000, platform: lng.NodePlatform}};
options.stage.window = {title: "Border example", fullscreen: false};
const app = new MyApp(options);

Save this into an mjs file (for example index.mjs).

Then you can run this using node --experimental-modules index.mjs.
