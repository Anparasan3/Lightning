Fire

.fire() executes a function in a child component.

Usage:

this.fire('methodName', arg1, arg2... argx)

Live Demo:

In this example, we will show a button that signals its parent to show/hide a message, using its parameters to change the font color as well.

class FireDemo extends lng.Application {
    static _template() {
        return {
            x: 20, y: 20,
            Message: {
                y: 80,
                type: ExampleComponent,
                message: 'This message will update\n in 3 seconds'
            }
        }
    }

    _init() {
        setTimeout(() => {
            this.tag('Message').fire('updateMessage', 'New Message')
        }, 3000)
    }
}

class ExampleComponent extends lng.Component {
    static _template() {
        return {}
    }
    _init() {
        this.updateMessage(this.message)
    }
    updateMessage(message) {
        this.patch({
            text: { text: message }
        })
    }
}

const app = new FireDemo({stage: {w: window.innerWidth, h: window.innerHeight, useImageWorker: false}});
document.body.appendChild(app.stage.getCanvas());
