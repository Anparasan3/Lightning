Tags

Tags are used as a reference to an object, inside a template.

static _template(){
    return {
        MyObject:{ x: 0, y: 0, w: 50, h: 50, rect: true }
    }
}

If you want to reference MyObject for instance, you can reference to said with this.tag(<object name>), as shown below.

const myObject = this.tag('MyObject');
