Image

Importing AppDefinition.js and setting a local image source:

If you want to use images stored locally, you have to import AppDefinition.js

    import Def from "<MyPath>/AppDefinition.js";

Then, when referring to an image, we use Def.getPath() to get the image source.

    MyLocalImage:{ src: Def.getPath('<MyPath>/MyImage.png')}

Setting a remote image source:

    MyRemoteImage:{ src: '<MyPath>/MyImage.png' }

Using local and remote images

In the following example we are going to display an image from a local and external source.

Live Demo:


class BasicUsageExample extends lng.Application {
    
    static _template(){
        return {
            MyLocalImage:{ x: 250, y: 200, src: Def.getPath('/Lightning/img/LngDocs_LilLightningIdle.png') },
            MyExternalImage:{ x: 250, y: 800, src: 'https://www.gstatic.com/webp/gallery3/1.png' }
        };
    }
}

const options = {stage: {w: window.innerWidth, h: window.innerHeight, clearColor: 0x00000000, canvas2d: false, useImageWorker: false}, debug: true};
const app = new BasicUsageExample(options);
document.body.appendChild(app.stage.getCanvas());
