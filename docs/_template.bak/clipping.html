Clipping

clipping allows for child-objects within a parent-object to be "clipped", when rendered outside the parent-object's width or height.

   MyObject:{ w: 100, h: 100, clipping: true,
        MyClippedObject:{ x: 50, y: 50, w: 100, h: 100, rect: true}
   }

Below an example with clipping turned on. Press left or right to move the green box.

class BasicUsageExample extends lng.Application {
    static _template() {
        return {
            //Try setting clipping to true or false, to see what happens.
            MyBlueCube:{ x: 100, y: 100, w: 100, h: 100, rect: true, clipping: true, color: 0xFF0034DD,
                MyGreenCube:{ x: 100, y: 50, w: 100, h: 100, rect: true, color: 0xFF24DD00 }
            }
        }
    }
          
     _handleLeft(){
        this.tag('MyGreenCube').patch({ smooth:{ x: 50 }});
     }
     
     _handleRight(){
        this.tag('MyGreenCube').patch({ smooth:{ x: 100 }});
    }
}


const options = {stage: {w: window.innerWidth, h: window.innerHeight, clearColor: 0x00000000, canvas2d: false, useImageWorker: false}, debug: true}
